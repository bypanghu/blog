Github转CDN高速链接

相信大家经常copy github仓库里面的文件，但是由于github的服务器在国外，经常被墙，导致网速慢到想打人
这篇博客教大家怎么快速下载github上的文件

准备好你的github下载链接,这篇教程使用的文件是我自己仓库看板娘的链接
https://github.com/bypanghu/kanbanniang
jsDelivr 是一个免费、开源的加速CDN公共服务，托管了许多大大小小的项目，可加速访问托管的项目目录或图片资源。如果你想加载github某个仓库中的某张图片，
使用方法官网有，但是有点不易懂，有的新手可能不太懂，
例如我上面链接的看板娘 https://github.com/bypanghu/kanbanniang 文件的autoload.js的cdn链接为
https://cdn.jsdelivr.net/gh/bypanghu/kanbanniang@master/autoload.js
比对着你就会发现 主地址为 https://cdn.jsdelivr.net/gh + user + 仓库名 @ master(分支) / 文件

刷新 Jsdelivr 缓存

你的 CDN 链接 想手动刷新时 把链接中的

https://cdn.jsdelivr.net/
替换成

https://purge.jsdelivr.net/
即可实时刷新
